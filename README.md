# KADRA

PWA zur Erstellung von Besprechungsprotokollen (AHO-Standard) für Immobilienprojekte.

**Live:** https://oscarlimaalphafox.github.io/KADRA/

## Stack

Vanilla JS · HTML5 · CSS3 · IndexedDB · PWA (Service Worker) · Tailwind CLI

## Build

```bash
npm install
npm run build
```

`src/main.css` → `css/style.css`
